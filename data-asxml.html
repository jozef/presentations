mapping any XML to Data

Data::asXML

mapping any Data to XML

more or less

serializing Perl data structures to XML

Supported types

  • SCALAR
  • ARRAY
  • HASH
  • REF
  • references

SCALAR

  • ->encode("abc")
  • <VALUE>abc</VALUE>

SCALAR undef

  • ->encode(undef)
  • <VALUE type="undef"/>

SCALAR binary

  • ->encode(chr(255))
  • <VALUE type="base64">/w==</VALUE>

REF

  • ->encode(\"abc")
  • <VALUE subtype="ref">abc</VALUE>

ARRAY

  • ->encode([ 1, 2, 3 ])
  • <ARRAY>
	<VALUE>1</VALUE>
	<VALUE>2</VALUE>
	<VALUE>3</VALUE>
</ARRAY>

HASH

  • ->encode({ 1 => 2, 3 => 4 })
  • <HASH>
	<KEY name="1">
		<VALUE>2</VALUE>
	</KEY>
	<KEY name="3">
		<VALUE>4</VALUE>
	</KEY>
</HASH>

references

  • ->encode([1,\%h,\%h])
  • <ARRAY>
	<VALUE>1</VALUE>
	<HASH>
		<KEY name="1">
			<VALUE>2</VALUE>
		</KEY>
	</HASH>
	<HASH href="*[2]"/>
</ARRAY>

see t/01_Data-asXML.t for more examples
